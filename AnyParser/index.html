Structure Any​Parser 

public struct AnyParser<Input, Output>: Parser

A type-erased parser of Output from Input.

This parser forwards its parse(_:) method to an arbitrary underlying parser having the same Input and Output types, hiding the specifics of the underlying Parser.

Use AnyParser to wrap a publisher whose type has details you don't want to expose across API boundaries, such as different modules. When you use type erasure this way, you can change the underlying parser over time without affecting existing clients.
