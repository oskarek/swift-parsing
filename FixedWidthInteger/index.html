parser(of:​is​Signed:​radix:​) 

@inlinable
  public static func parser<Input>(
    of inputType: Input.Type = Input.self,
    isSigned: Bool = true,
    radix: Self = 10
  ) -> Parsers.IntParser<Input, Self>

A parser that consumes an integer (with an optional leading + or - sign) from the beginning of a collection of UTF-8 code units.

var input = "123 Hello world"[...].utf8
let output = Int.parser().parse(&input)
precondition(output == 123)
precondition(Substring(input) == " Hello world")

Parameters

The collection type of UTF-8 code units to parse.

radix Self

The radix, or base, to use for converting text to an integer value. radix must be in the range 2...36.

Returns
