Structure Always 

public struct Always<Input, Output>: Parser

A parser that always succeeds by returning the provided value, but does not consume any of its input.

var input = "Hello"[...]
let output = Always(1).parse(&hello)
precondition(output == 1)
precondition(input == "Hello")
