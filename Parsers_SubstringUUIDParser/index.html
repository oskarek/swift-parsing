Structure Parsers.​Substring​UUIDParser 

public struct SubstringUUIDParser: Parser

A parser that consumes a UUID from the beginning of a substring using a UTF-8 parser.

You will not typically need to interact with this type directly. Instead you will usually use UUID.parser(), which constructs this type.
