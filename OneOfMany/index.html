Structure One​OfMany 

public struct OneOfMany<Upstream>: Parser where Upstream: Parser

A parser that attempts to run a number of parsers till one succeeds.

This is a more performant version of Parser/orElse(_:) that can be used when Upstream is the same type.

For example, OneOfMany can capture the work of a number of parsers that do similar work and are thus all Parsers.Map<StartsWith<Input>, Output>:

enum Currency { case eur, gbp, usd }

let currency = OneOfMany(
  StartsWith("€").map { Currency.eur },
  StartsWith("£").map { .gbp },
  StartsWith("$").map { .usd }
)
